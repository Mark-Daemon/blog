STL COLOUR PREVIEW TOOL
2025-04-13

Link to the tool here.

The tool assumes you will print with a first layer height of 0.16mm and a general layer height of 0.08mm.
You can adjust the colours applied by layer and how many are used.
The approach works well with an image with a good range of tones or applying a mosaic effect to improve the blending between layers.

The script runs within your browser. If you wish to edit it, you can download the page source code and serve it locally with python -m http.server.
