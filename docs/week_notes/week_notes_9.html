Personal Metrics

Clojure-Script Integration

As I have continued looking for excuses to write Clojure, I integrated Clojure-Script into the site. The process was simple - I can now write Clojure-Script code in dedicated files and have them compiled alongside the overall static-site generation into Javascript. However, it is slower than I had hoped.

I have some thoughts on implementing a caching solution to speed it up. Along the lines of storing the compiled JS alongside the last-modified date of the .cljs file and only re-compiling as needed. Perhaps next week.
